Çalışması için gmailden APP PASSWORD almanız lazım. Güvenli olmadığından dolayı mail adresinizin kendi şifresi ile girmenize izin vermiyor. 

https://myaccount.google.com/apppasswords
^
Adresine girip (eğer 2 aşamalı güvenlik özellğiniz açıksa) uygulama adı giriniz.
Sonrasında karşınıza çıkan kelime bütününü yazdığım uygulamada mail gönderme aşamasında size şifre sorunca oraya giriniz.
Bu şifre, şifreye sahip olan kişinin sadece mail üzerinden mesaj atabilmesini sağlamakta olup tamamen güvenilirdir.
Şifreyi elinde bulunduran bir kişi hesaba erişim sağlayamaz.
